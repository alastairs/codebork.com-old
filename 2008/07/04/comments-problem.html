Comments Problem

July 4, 2008 - 1 minute read -

So it seems there’s currently a problem with the comments’ system. First of all, DON’T PANIC! Your awesome comments are being posted correctly, and are making it through to the moderation queue fine (or being posted if you’ve got an account). The bad news is the Akismet anti-spam module I’ve installed is in beta for Drupal 6 at the moment, and an error in the module is causing the screen to go blank when you hit “Save” to publish your awesome comment.

If you want the technical detail, a function that this module called has been removed from the Drupal API in Drupal 6. There’s more information available on the bug report over at Drupal.org.
